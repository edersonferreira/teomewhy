# Teomewhy CLI

# Base

This project is a copy of [Oloquinho Meu](https://github.com/oloquinho/oloquinho) with Teomewhy audio Kappa Kappa Kappa

## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g teomewhy

## Running

With npm:

    npm install -g teomewhy
    teomewhy

Npx:

    npx teomewhy

Node script:

    import teomewhy from 'teomewhy';

    teomewhy();
